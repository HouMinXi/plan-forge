# Host-Search Injection Eval

One-shot exploratory eval to answer: When deepseek/mimo are given the
host's exhaustive search evidence, how accurate is their citation
resolvability judgment vs the host judging the same evidence?

**This is a prompt-level exploratory eval, not a CI regression gate.**
It calls provider APIs directly; it does not drive the real
`api.check()` / G8 gate. A pytest-based regression test that drives
`check()` on these fixtures is a separate future task.

## Files

- `runner.py`: The eval script. Reads the canonical 12-fixture set
  from `tests/fixtures/g8_hostsearch_fixtures.json` (single source of
  truth; no local fixtures copy here).
- `report.md`, `report.json`: Recorded baseline from the original run.
  These are historical artifacts and are not regenerated by CI.

## Fixtures

The 12 test cases live at `tests/fixtures/g8_hostsearch_fixtures.json`
(canonical). That file is not duplicated here.

## Usage

```bash
cd /path/to/repo
uv run python eval/host_search/runner.py
```

Requires API keys in env or pass:
- `PLAN_FORGE_DEEPSEEK_API_KEY` or `pass show api/deepseek`
- `PLAN_FORGE_MIMO_API_KEY` or `pass show api/mimo`

Running this script costs real API credits. The recorded baseline in
`report.md` and `report.json` captures the original run results.

## Conditions

- `ds_evidence`: deepseek with injected host search evidence
- `ds_noevidence`: deepseek without evidence (control)
- `mimo_thinkon_evidence`: mimo (thinking on) with injected evidence
- `mimo_thinkon_noevidence`: mimo (thinking on) without evidence
- `mimo_thinkoff_evidence`: mimo (thinking off) with injected evidence
- `mimo_thinkoff_noevidence`: mimo (thinking off) without evidence
- `host`: host's own verdict from fixtures

## Grading

- `CORRECT`: matches ground truth
- `CONSERVATIVE`: UNCERTAIN (honest hedge, surfaces to human)
- `DANGEROUS`: false positive or false negative
- `fabricated_resolution`: not_found citation answered RESOLVED_VIA_SEARCH
